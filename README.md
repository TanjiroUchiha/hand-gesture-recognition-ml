# Real-Time Hand Gesture Recognition using Python & MediaPipe

This project implements a real-time hand gesture recognition system using OpenCV, MediaPipe, and Machine Learning (Random Forest).

The pipeline includes:

- Collecting hand landmark data via webcam
- Training a classifier on custom gestures
- Predicting gestures live using the trained model
